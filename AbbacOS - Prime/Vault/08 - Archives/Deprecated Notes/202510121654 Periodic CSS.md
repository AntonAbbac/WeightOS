### Funções de códigos

- “:root” — É como uma caixa de comando já pré-definidos com um ou mais comandos atribuídos.

- “.exemplo” — “.” Serve para definir um comando para a edição do mesmo

- “--exemplo” — “--” Serve para definir uma argumentação para determinado resultado.

- “color:” — Define a cor de determinado elemento.

- “background-color:” — Define a cor de fundo de determinado elemento.

- “text-align:” — Define o alinhamento de determinado texto.

- “font-size:” — Define o tamanho de determinada fonte.

- “font-family:” — Define o formato da fonte de determinado texto.

- “padding:” — Define o preenchimento de determinado elemento.
	 - “padding-top” — Define o preenchimento superior.
	 - “padding-left” — Define o preenchimento à esquerda.
	 - “padding-right” — Define o preenchimento à direita.

- “border-color:” Define a cor de borda de um determinado elemento.

- “font-weight:” — Define a espessura de um determinado elemento.

- “magin-bottom:” — Define a distância da margem inferior em relação ao espaçamento.

- “width:” — Define a largura de determinado elemento.

- “border:” — Define o tipo de borda.
  
- “border-radius:” — Define o formato da borda.
  
- “border-width:” — Define a espessura da borda.

- “word-wrap:” — Define a quebra automática de linha.

- “margin-top:” — Define a distância da margem superior em relação ao espaçamento.

- “margin-left:” — Define a distância da margem esquerda em relação ao espaçamento.

- “margin-right:” — Define a distância da margem direita em relação ao espaçamento.

- “display:” — Muito subjetivo.
