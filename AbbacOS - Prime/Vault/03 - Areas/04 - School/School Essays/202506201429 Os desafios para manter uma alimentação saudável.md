Oque é compõe uma alimentação saudável?

Muitas vezes queremos levar nossa vida a um patamar acima, melhorando nossos hábitos de higiene, cuidados e alimentação. Porém, isso se torna um desafio para cada um, sendo uma tarefa de complexidade pessoal alta. Muitos não sabem, mas a alimentação é um dos pilares básicos mais importantes que o ser humano deve possuir em sua vida. Mas o ponto mestre de tudo antes de se dizer o que você deve fazer para manter uma alimentação saudável, devemos nos perguntarmos o que não compõe uma alimentação saudável.

O que deve ser evitado em uma alimentação saudável?

Muitos alimentos que se denominam "Fit" nas prateleiras dos mercados pregam um senso de liberdade existencial, como se você se livrasse da preocupação de controlar e medir os produtos que consome. Mas não é bem assim a realidade. Muitas marcas famosas adotam a estratégia de se auto proclamar "fitness", mesmo tendo uma tabela nutricional parecida ou igual a outras vistas como "JunkFood" ou porcarias ultra-processadas. É importante ter conhecimento nutricional básico para ter certeza do que se consome, e mesmo assim pode ser uma opção errada para seu objetivo de se alimentar de forma consciente.

Como se alimentar de forma saudável e se manter firme contra adversidades?

Para nós sabermos quanto uma pessoa deve consumir em macro e micronutrientes é necessário marcar uma consulta com seu nutricionista pessoal, para ele mesmo fazer um cálculo em cima das suas funções metabólicas. Porém, há um jeito de fazer isso em casa em base no que é visível, mesmo não sendo tão profundo quanto o auxílio de um profissional, mas ainda é melhor que a ausência deste conhecimento essencial. Para sabermos quantas calorias devemos consumir em uma média superficial, usamos a equação de Mifflin-St Jeor para calcular a TMB(Taxa metabólica Basal).
