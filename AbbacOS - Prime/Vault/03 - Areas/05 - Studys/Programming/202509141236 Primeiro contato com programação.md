## Funções de comandos aprendidos

```
**<>** — # Em HTML esse símbolo significa a abertura de uma função, e o que você escrever daquilo em diante será notado pelo seu navegador. É a abertura da estrutura o subestrutura.

</> — # É o mesmo príncipio do <> , porém ele define o fechamento de uma estrtura como um todo.

<header> — # É o cabeçalho da sua estrutura, o que abriga funções escondidas.

<title> — # É o título da sua estrutura.

<style>(CSS) — # É a estilização da sua estrutura, que altera aparência de comandos de estrutura do HTML> É basicamente o CSS.
	
	 - background-color: “Color“; — # Altera a cor de determinada função.
	 - font-size: ??px; — # Muda o tamanho de determinada fonte.
	 - width: ??px; — # Altera a largura.
	 - height: ??px; — # Altera a altura.
	 - cursor: pointer; — # Muda a ação do ponteiro do mouse ao passar por cima da propriedade “botton”.
	
	

<body> — # É o corpo do seu texto, o que ficará visível no seu site.

<footer> — # Onde se abriga comandos de programação, como funções e outros comandos.

<img src= "“ — # AdicionaUma imagem a base de links entre os “”.
```
